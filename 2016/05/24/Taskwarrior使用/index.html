Taskwarrior使用

  • tool

这是一个在终端下的task,用来标记todo list挺不错的.

虽然atom的很多插件都能满足,但是有时候atom的速度太慢,固定硬盘受不了这速度.

所以找了个用终端来记录的,虽然emacs的org就能很好的,但是无奈emacs苦手╮(╯▽╰)╭

##mac 下的安装

##task的使用

新增任务

task add 任务描述

查看任务列表

task list

输出

可以对id来对任务进行操作

这只能看到添加的任务,但是done和delete的任务看不到

查看所有任务(包括删除和完成的)

task all

查看任务详细信息

task id info

传入task id或者uuid可以查看该任务的详细信息

开始一个任务

task id start

任务默认start状态是没有的

为任务添加所属

task 1 modify project:work

为任务添加一个所属,为 project:work

task list 输出

也可以在添加任务的时候添加所属

task add project:work 添加所属任务

删除任务

task 1 delete

删除任务的时候会提醒你是否删除

标记任务完成

task 1 done

为任务修改属性

修改所属项目

task 1 modify project:home

为其添加标签

task 1 modify +home

为其删除标签

task 1 modify -home

为其添加结束时间

task 1 modify due:today

due的时间设置很多可以直接设置YYYY-MM-DD设置日期

也可以直接设置YYYY-MM-DDThh:mm:ss设置具体的时间

在新建任务的时候可以直接设置这些属性

task add 新增任务 project:home +home due:2016-06-11T23:12:00

查看该任务的info如下

为任务添加备注

task id annotate 描述内容

为任务添加描述内容

task id append 最追加的内容,在现有描述的后面

task id prepend 最追加的内容,在现有描述的前面

查看日历

task calendar
